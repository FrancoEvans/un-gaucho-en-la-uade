##### Arreglos										

Reserva un bloque contiguo de memoria. Es obligatorio definir su tamaño antes de usarlo.

##### Métodos (Bloques de Acción)

Equivalentes a las funciones / procedimientos. Tienen un tipo de retorno estricto. Si no devuelven nada, se declaran como "void".		

Parámetros ---> Procesamiento ---> Retorno

##### Los Planos (Clases y Objetos)

Una clase es una plantilla estructural que agrupa variables (estado) y métodos (comportamiento).					# Bit: unidad más pequeña de información.

Objeto: instancia de una clase, único e irrepetible.

##### El paquete básico y la consola

Java se organiza en bibliotecas llamadas paquetes. Usamos la notación de punto (.) para navegar por esta jerarquía e invocar métodos. 

java.lang > System > out 

##### Valor vs Referencia

###### Por valor					Por referencia	

\_ Se pasa una copia del dato original al método.		\_ Se pasa la dirección de memoria.

\_ El método trabaja sobre la copia. 				\_ El método interactúa directamente con el dato original y puede alterarlo permanentemente.

\_ El dato original está a salvo y no muta.

##### TDAs (Tipos de Datos Abstractos)

El objetivo de la cursada será usar la robusta estructura de Java para crear Tipos de Datos Abstractos.
